We want to write a follow up LaTeX papr to article.tex and article2.tex about shrodinger's equations and the conflicts with general relativity and quantum physics.

Shrodinger's equations have a time free version and a time dependent version. But general relativity deals with spacetime, which means there's a fundamental mismatch in these two frameworks which cannot been easily reconciled.

Also quantum physics raise issues about the wave function collapse and what is an observer, which was slightly touched in article2.tex.

We claim that only dealing with the lightlike graph simplifies the maths enough that we can start thinking about how to resolve all these issues to build an unified theory of physics.
